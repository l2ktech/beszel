## 🔧 本地服务器连接修复

### 问题说明
本地服务器显示红色（未连接）是因为 Beszel 使用 SSH 方式让 Hub 连接到 Agent，而不是 Agent 主动连接 Hub。

### 正确的连接方式

**Beszel 架构说明**:
- Agent 运行 SSH 服务器（监听端口 45876）
- Hub 通过 SSH 连接到 Agent
- 使用 SSH 密钥进行身份验证

## 🌐 如何添加其他服务器

### 远程服务器连接步骤

#### 1. 准备目标服务器
确保目标服务器满足以下条件：
- 已安装 Docker
- 可以通过 SSH 访问
- 有 root 或 sudo 权限

#### 2. 选择连接方式

**方式一：使用 Web 界面（最简单）**

1. **访问 Beszel**
   ```
   URL: http://localhost:38005
   ```

2. **点击 "Add System"**

3. **填写服务器信息**
   - **System Name**: 给服务器起个名字（如：生产服务器）
   - **Host/IP**: 服务器 IP 地址（如：192.168.1.100）
   - **SSH Port**: SSH 端口（默认 22）
   - **SSH User**: SSH 用户名（如：root）

4. **配置 SSH 密钥**
   - 选择使用公钥或密码认证
   - 如果使用公钥，将本地公钥添加到目标服务器

5. **复制安装命令**
   - 添加系统后，Beszel 会显示安装命令
   - 复制该命令

6. **在目标服务器执行安装命令**
   ```bash
   # SSH 连接到目标服务器
   ssh user@your-server-ip

   # 执行安装命令（从 Web 界面复制）
   curl -sSL https://beszel.dev/install | sh
   ```

7. **验证连接**
   - 返回 Beszel Web 界面
   - 服务器状态应该变为绿色

**方式二：手动安装 Agent（推荐用于生产环境）**

1. **在目标服务器创建目录**
   ```bash
   mkdir -p ~/beszel-agent && cd ~/beszel-agent
   ```

2. **创建 docker-compose.yml**
   ```yaml
   services:
     beszel-agent:
       image: henrygd/beszel-agent
       container_name: beszel-agent
       restart: unless-stopped
       network_mode: host
       volumes:
         - ./beszel_agent_data:/var/lib/beszel-agent
         - /var/run/docker.sock:/var/run/docker.sock:ro
       environment:
         PORT: 45876
         KEY: 'ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIAq8t3HXdYTMp8kWBwUuCbuezv/i74jcGgTlWvSaKJVA'
         # FILESYSTEM: /dev/sda1 # 根据实际情况修改
   ```

3. **启动 Agent**
   ```bash
   docker compose up -d
   ```

4. **在 Beszel Hub 添加服务器**
   - 访问 http://localhost:38005
   - 点击 "Add System"
   - 填写服务器信息：
     - System Name: 服务器名称
     - Host/IP: 服务器 IP
     - SSH Port: 45876（agent 监听的端口）
   - 使用 SSH 密钥认证

5. **配置 SSH 密钥**
   - 将 Hub 的公钥添加到目标服务器的 `~/.ssh/authorized_keys`
   - 或使用密码认证

**方式三：使用 WinSCP 添加 SSH 密钥**

1. **使用 WinSCP 连接到目标服务器**
   - 下载 WinSCP: https://winscp.net/

2. **复制公钥到目标服务器**
   - 打开 `/Users/wzy/projects/08-Beszel/keys/beszel_agent.pub`
   - 复制公钥内容

3. **添加到 authorized_keys**
   ```bash
   # 在目标服务器上
   mkdir -p ~/.ssh
   chmod 700 ~/.ssh
   echo "ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIAq8t3HXdYTMp8kWBwUuCbuezv/i74jcGgTlWvSaKJVA" >> ~/.ssh/authorized_keys
   chmod 600 ~/.ssh/authorized_keys
   ```

4. **在 Beszel 添加服务器**
   - 访问 http://localhost:38005
   - 点击 "Add System"
   - 填写服务器信息
   - 选择 SSH 密钥认证

## 🔍 故障排查

### 本地服务器无法连接

**检查 Agent 是否运行**
```bash
cd /Users/wzy/projects/08-Beszel
docker compose ps beszel-agent
```

**检查 Agent 日志**
```bash
docker compose logs beszel-agent --tail=50
```

**检查 SSH 端口**
```bash
lsof -i :45876
```

**测试 SSH 连接**
```bash
ssh -i /Users/wzy/projects/08-Beszel/keys/beszel_agent \
  -p 45876 -o StrictHostKeyChecking=no \
  -o UserKnownHostsFile=/dev/null \
  localhost echo "SSH连接成功"
```

### 远程服务器无法连接

**检查网络连通性**
```bash
ping remote-server-ip
telnet remote-server-ip 45876
```

**检查防火墙**
```bash
# 在远程服务器上
sudo ufw status
sudo ufw allow 45876/tcp
```

**检查 Agent 日志**
```bash
# 在远程服务器上
docker logs beszel-agent --tail=50
```

**检查 Hub 日志**
```bash
cd /Users/wzy/projects/08-Beszel
docker compose logs beszel --tail=50
```
