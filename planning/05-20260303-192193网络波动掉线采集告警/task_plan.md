- [2026-03-03] 当前实现仅写入 `info.z192/z193`，尚无抖动/掉线状态字段与告警状态机。

- [2026-03-08] `beszel_data/data.db` 执行 `PRAGMA quick_check` 返回 `database disk image is malformed`，`system_stats` 索引与空闲页损坏；运行中 `1m/zt1m` 最新样本停在 2026-03-05，说明当前停更首先是数据库损坏而非前端渲染缺失。
